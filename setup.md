# Setup del proyecto

## Reto Integrador - Unidad 1

En este documento se explica cómo preparar el entorno para ejecutar correctamente los notebooks del proyecto.

## Requisitos

Antes de comenzar es necesario tener instalado:

- Python 3.11 o superior.
- Visual Studio Code.
- La extensión Jupyter para Visual Studio Code.

## Instalación de librerías

Abrir una terminal y ejecutar el siguiente comando para instalar las librerías utilizadas durante el desarrollo del reto:

```bash
pip install numpy pandas matplotlib jupyter
```

## Abrir el proyecto

1. Abrir Visual Studio Code.
2. Seleccionar **Archivo → Abrir carpeta**.
3. Abrir la carpeta **reto_u1**.

## Ejecutar los notebooks

Una vez abierto el proyecto:

1. Abrir el notebook que se desea ejecutar.
2. Seleccionar el Kernel de Python donde se instalaron las librerías.
3. Ejecutar **Kernel → Restart & Run All**.

Los notebooks incluidos en el proyecto son:

- `01_numpy.ipynb`
- `02_sql_discovery.ipynb`
- `03_pandas_pipeline.ipynb`

## Observaciones

Para este reto se utilizó una copia del archivo de datos llamada **ventas_techventas_limpio.csv**, ya que el archivo original presentaba algunos errores de formato que impedían su correcta lectura. El archivo original se conservó sin modificaciones.
